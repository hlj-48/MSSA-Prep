# MSSA-Prep
This repository serves to document the journey of acquiring technical skills and understanding material for the eventual application to the Miscrosoft Software and Systems Academy (MSSA)

## Learning Path

### Phase 1 - Foundations
- Create GitHub account and first README
- Learn Networking basics
- Study SEC+ principles utilizing Microsoft Learn and other resources
- Get familiar with the Microsoft ecosystem
- Complete TryHackMe labs
- Learn PowerShell and Python
- Create a home lab with virtual machines
- Obtain CompTIA SEC+ certification

### Phase 2 - Experience
- Apply for entry level IT jobs with focus on cybersecurity
- Work for at least a year 
- Complete independent projects to further mastery
- Gain skills and possible promotion

### Phase 3 - MSSA Application Launch
- Review LinkedIn, resume, and GitHub accounts
- Attend MSSA info session
- Apply for the next MSSA cohort
